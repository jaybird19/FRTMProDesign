Detailed Description

Main OpenGL screen showing the 3D World and coverage of all Camera instances.

This class is used to:

  1. Display the main 3D Model that represents the world in which Camera(s) are added.
  2. Show Camera(s) coverage.
  3. Show the positions of Camera(s).
  4. Show the Labels of Camera(s).
  5. Place Camera(s) in the 3D world by dragging the camera icon into MainView.
  6. Change the position of Camera(s).
