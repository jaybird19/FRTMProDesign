Detailed Description

Camera Label(s) Container Manager.

Provides a container wrapper to store Label(s) of Camera(s). Adds extra features in order to manage the Label(s) in the MainView QGLWidget.

This class is used to:

  1. Add and delete Label(s).
  2. Draw Label(s) in MainView.
