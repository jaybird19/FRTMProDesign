Detailed Description

Manages a .obj 3D model to draw on an OpenGL screen.

This class is used to:

  1. Draw a .obj 3D model in a OpenGL window.
  2. Load Vertex attributes (position, normal, tex coordinates) into Vertex Buffer Objects and associates them with a GLSLProgram.
  3. Share VBO resources with more than one OpenGL context by generating a unique VAO per context.
