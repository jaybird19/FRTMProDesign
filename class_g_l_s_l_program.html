Detailed Description

Provides necessary functions to build a program object consisting of a vertex and fragment shader.

This class is used to:

  1. Build a GLSL Shader Program through linking and compiling vertex and fragment shaders.
  2. Add attributes and uniforms to the program object.
  3. Query attributes and uniforms location in the OpenGL context.
  4. Enable and disable the program object for rendering.

This is an abstract class. Specific GLSL programs are built by inheriting from this class and using its protected functions. Sample usage:

loadShader(GL_VERTEX_SHADER, "shader.vert");
loadShader(GL_FRAGMENT_SHADER, "shader.frag");
CreateAndLinkProgram();
use();
//Add attributes
addAttribute("attribute1");
addAttribute("attribute2");
//Add uniforms
addUniform("uniform1");
addUniform("uniform2");
//Specify the value of uniform variables for the current program object
glUniform1i((*this)("uniform1"), 2);
unUse();
