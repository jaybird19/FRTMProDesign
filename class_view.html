Detailed Description

Manages the position and view direction of an OpenGL window's View.

Provides implementation for travelling in a 3D world via mouse and keyboard inputs.

This class is used to:

  1. Travel in a 3D world with keyboard (WASD) inputs.
  2. Alter view direction with mouse (left click).
  3. Zoom in/out with mouse (middle click).
  4. Transition from one position to another in a 3D world.
