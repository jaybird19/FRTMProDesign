Detailed Description

ZBuffer texture container/manager for Camera(s)

This class stores and manages resources for drawing Z-Buffer (i.e. depth buffer) textures for each Camera in the Cameras class.

The Z-Buffer textures are used to accomplish shawdow mapping when displaying the Cameras' coverage.

Two textures are drawn for each Camera:

  1. Standard depth texture for the Camera's View.
  2. 180 FOV depth texture that is used to show Full Coverage of PTZ Cameras.

This class is used to:

  1. Add and delete Z-Buffer textures.
  2. Draw Z-Buffer textures on a dedicated framebuffer.
  3. Query generated Z-Buffer textures for use in shaders.
